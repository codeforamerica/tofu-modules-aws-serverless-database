# Changelog

## 1.3.0 (2025-05-09)

### Feat

- Reduce length of the monitoring role name with new (optional) `short_*` inputs. (#18)

### Fix

- Added default value for `cluster_parameters`. (#16)

### Feat

- Allow the number of instances to be configured. (#7)
